Named Transitions

November 17, 2025

Elemente mit gleichem view-transition-name morphieren automatisch zwischen Seiten:


article-title { view-transition-name: title }
.detail-title { view-transition-name: title }

Wichtige Limitation: Jeder view-transition-name muss pro Seite eindeutig sein. Für Listen mit mehreren Elementen braucht man dynamische IDs wie post-123, post-124, etc.

Custom Animationen sind mit ::view-transition-old() und ::view-transition-new() moeglich. So kann man Fade, Slide, Scale oder jede andere CSS-Animation definieren.
