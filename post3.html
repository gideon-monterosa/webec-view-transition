Browser Support (Nov 2025)

November 17, 2025

Die View Transitions API hat unterschiedliche Support-Levels je nach Feature:

  • Cross-Document Navigation: Chrome 126+, Safari 18.2+
  • Same-Document Transitions: Chrome 111+, Safari 18+, Firefox 144+
  • View Transition Classes: Chrome 125+, Safari 18.2+, Firefox 144+

Zero JavaScript nötig, Progressive Enhancement garantiert, dass die Seite überall funktioniert. In Browsern ohne Support gibt es einfach einen normalen Seitenwechsel.
